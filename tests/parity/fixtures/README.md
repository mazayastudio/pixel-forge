# Parity test fixtures

Bootstrap fixture `blank_16x16.aseprite` is generated by:

```powershell
python tests/parity/run_parity.py --generate-fixtures
```

Additional files should be exported from desktop **Aseprite v1.3.17.x** on MAZAYA-STUDIO.

| File | Purpose | Status |
|------|---------|--------|
| `blank_16x16.aseprite` | Minimal indexed sprite | Auto-generated |
| `tags_walk_cycle.aseprite` | Tags with ping-pong + repeat | Manual export |
| `tilemap_manual.aseprite` | Manual tilemap layer | Manual export |
| `linked_cels.aseprite` | Linked cel animation | Manual export |
| `slices_ninepatch.aseprite` | 9-slice regions | Manual export |

Run: `python run_parity.py --list`
