## Parity test harness

- Fixtures: `tests/parity/fixtures/*.aseprite`
- Golden images: `tests/parity/golden/` (generated by Aseprite CLI on MAZAYA-STUDIO)
- Runner: `tests/parity/run_parity.py` — invokes WASM + NDK builds
- Report: `tests/parity/output/report.json`

---

## Exception process

1. File issue with matrix row ID (e.g. `T05`)
2. Document behavioral delta vs Aseprite 1.3.17.x
3. Assign fix phase and owner
4. Do not ship Phase 0 with undocumented P0 failures
